TrimTime is a simple salon slot booking web application built using HTML, CSS, and JavaScript.

## Features
- Login page with input validation
- City-wise salon selection
- Salon list based on selected city
- Time slot selection
- Clean and user-friendly UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Pages
1. Login Page
2. City & Salon Selection Page
3. Time Slot Selection Page

## How to Run
Open `index.html` in a browser or access the live site using GitHub Pages.

## Project Status
This project is created as a beginner-level web development project.
